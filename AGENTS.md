# AGENTS

このリポジトリの作業を始める前に、必ず本ファイル（AGENTS.md）を読むこと。

## 作業ルール
- 追加で修正を行う場合は、必ず最初に本ファイル（AGENTS.md）の仕様書を更新すること。
- ここで扱う内容はサンプルサイト用の仮情報を含む。
- 公開前にオーナー確認が必要。

## 仕様書（統合版）

### 1. 目的と前提
- Bar the RING（新橋）のサンプルHPを制作する。
- 公式情報が未確定のため、第三者情報とユーザー指定情報を根拠に構成する。
- 参考サイトの雰囲気・構成を踏襲しつつ、文章・ビジュアルはオリジナルで作る。
- サンプルサイトである旨を明記する。

### 2. 参照情報（根拠）
- 住所・営業時間・定休日・設備（カラオケ/ダーツ）についての第三者記載。
- 立地（内幸町/新橋アクセス）・2025年オープン・大人向け空間・遊び要素・テキーラに注力等の第三者記載。
- オーナー設定は「保田さんがプロボクサーのセカンドキャリアとして開店」というユーザー指定。
- Instagramアカウント: barthering419（ユーザー指定）

### 3. 情報設計（ページ構成）
1. TOP（ロゴ・住所帯・ナビ・ヒーロー画像）
2. INFORMATION（料金/営業時間/定休日/予約手段）
3. NEWS（更新情報のカード型リスト）
4. CONCEPT（オーナーストーリー＋店の価値訴求）
5. GALLERY（店内/VIP/メニュー等の写真グリッド）
6. ACCESS（住所・TEL・地図）
7. LINK（Instagram/予約導線）

### 4. コンテンツ方針
- トーンは上質で落ち着いた大人向け。
- 文章は「サンプル」「参考情報に基づく」前提で記載。
- 料金・営業時間・住所は第三者情報をベースに明記。
- 予約導線は「LINE/電話/Instagram DM（サンプル）」で構成。
- Instagramは barthering419 を表示。

### 5. 画像・アセット対応
- assets/view.jpeg = 店内の雰囲気
- assets/biproom.jpeg = VIP席
- assets/logo.jpeg = ロゴ
- assets/menu1.png = 通常メニュー表
- assets/menu2.jpeg = 飲み放題メニュー表

### 6. デザイン方針
- 参考サイトの「白ベース＋上品な余白＋細い罫線」方向。
- タイポはクラシック×和文の組み合わせ。
- ギャラリーは多枚数グリッドで見せる。
- NEWSは画像＋日付＋タイトルのカード型。

### 7. 実装対象ファイル
- index.html
- styles.css
- script.js（必要最小限のスクロール演出のみ）

### 8. 注意・免責
- 公式情報ではないため、公開時は必ず内容確認が必要。
- 参考サイトの構成は踏襲するが、テキストや見せ方は独自にする。
