## 仕様書（統合版）

- オーナー安田勝也氏は元プロボクサー（WBOアジアパシフィックライト級王者）。セカンドキャリアとして2025年2月にオープン。

### 3. 情報設計（ページ構成）
1. HEADER（固定ヘッダー: ロゴ左 + ナビ右(PC) / ハンバーガー(SP)）
2. HERO（view.jpeg全画面背景 + ロゴ円形表示 + タグライン）

### 4. コンテンツ方針
- トーンは上質で落ち着いた大人向け。
- 文章は「サンプル」「参考情報に基づく」前提で記載。
- 料金・営業時間・住所は第三者情報をベースに明記。
- 予約導線は「LINE/電話/Instagram DM」で構成。
- Instagramは barthering419 を表示。
- バイリンガル見出し: 英語タイトル（大）＋ 日本語サブタイトル（小）の統一パターン。

- assets/view.jpeg = 店内の雰囲気（ヒーロー背景 + ギャラリー）
- assets/biproom.jpeg = VIP席（コンセプト + ギャラリー）
- assets/logo.jpeg = ロゴ（ヘッダー・ヒーロー・フッター）
- assets/menu1.png = 通常メニュー表（ギャラリー）
- assets/menu2.jpeg = 飲み放題メニュー表（ギャラリー）
- assets/LINE.jpg = オーナーのLINEのQRコード（RESERVATIONセクション）
- assets/Instagram.jpg = InstagramのQRコード（RESERVATIONセクション）

### 6. デザイン方針
- **ダークテーマ**: 黒背景(#0a0a0a) + 金色アクセント(#c9a96e) + 白テキスト
- セクション交互背景(#0a0a0a / #141414)で視覚的区切り
- カード背景: #1a1a1a
- フォント: Cormorant Garamond（英語見出し）+ Noto Sans JP（日本語本文）
- clamp()によるフルイドタイポグラフィ
- モバイルファースト: 基本(SP) → 640px(タブレット) → 1024px(PC)
- 固定ヘッダー: 透明 → スクロールで背景色+blur付加
- ハンバーガーメニュー: フルスクリーンオーバーレイ、X変形アニメーション
- ヒーロー: 100vh、Ken Burnsエフェクト
- ギャラリー: hover時zoom(1.05)
- スクロールアニメーション: IntersectionObserverによるfade-in + slide-up
- LINE QRコード: 白パディングフレームで読み取り可能に
- ギャラリーライトボックス: クリックでフルサイズ表示
- セクション見出し(h2): 文字下部に金色(#c9a96e)グラデーションのフットライト風装飾（::after疑似要素、z-index:-1で文字背後）
